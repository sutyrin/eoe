# AGENTS (games)

Общие правила для всех игр. Локальные AGENTS.md в конкретной игре имеют приоритет.

## Общие принципы

- Ориентация: портрет, управление в нижней трети экрана.
- Пошаговый геймплей без давления на реакцию.
- Любое визуальное событие должно быть следствием изменения state.

## Тестовая пирамида (кратко)

- Юнит: 60–80%
- Game API: 15–30%
- E2E UI: 5–10%

## Что писать где

- Локальные правила и TODO — в `games/<game>/AGENTS.md`.
- Общие решения по игровым паттернам — сюда.
- Архив задач — `docs/archive.md`.
