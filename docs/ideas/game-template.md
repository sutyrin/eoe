# Шаблон игры: принципы и идеи

Этот документ — для генерации идей и дизайна новых игр. Не использовать при реализации конкретных игр.

## Общие принципы шаблона

1.  **Пошаговость (Turn-based):**
    * Игровой мир замирает в ожидании действий игрока.
    * Нет таймеров обратного отсчета или давления на реакцию.
    * Игрок (и AI-агент) может думать сколько угодно перед ходом.

2.  **Управление (Mobile-first):**
    * Ориентация экрана: вертикальная (portrait).
    * Зона контроля: строго в нижней трети экрана (удобно для большого пальца).
    * Количество кнопок: минималистичное, обычно 2 или 3.

3.  **Прогрессия:**
    * Движение преимущественно направлено "вверх" (или вперед по вертикали).
    * Игрок видит ближайшее будущее (следующие несколько рядов/тайлов) в верхней части экрана.
    * Прошлые шаги уходят вниз и исчезают.

4.  **Визуальный стиль:**
    * Живая картинка: несмотря на пошаговость, мир не статичен. Используются idle-анимации героя, врагов, фоновые эффекты.
    * Четкая сетка (grid): игровое поле дискретно, что упрощает визуальное считывание.

5.  **Agent-Friendly:**
    * Состояние игры полностью описывается JSON-объектом.
    * Любое визуальное событие является следствием изменения state.

## 5 механик для данного шаблона

1. **"Бесконечный выбор пути" (Branching Paths):**
   * Классический раннер, но пошаговый. На пути встречаются препятствия (камни, ямы).
   * Игрок выбирает полосу движения (лево/центр/право), чтобы избежать урона и собрать монетки.

2. **"Тактическое восхождение" (Resource Management):**
   * Каждый шаг тратит ресурс (выносливость/топливо).
   * На поле разбросаны восполняющие предметы и враги.

3. **"Камень-Ножницы-Бумага" (Encounter Combat):**
   * Путь блокируют враги разных типов (Красный, Зеленый, Синий).
   * У игрока есть 3 кнопки-стойки. Нужно выбрать правильную стойку перед шагом.

4. **"Сокобан-лайт" (Pushing):**
   * На пути встречаются блоки, которые можно толкать перед собой.
   * Задача: толкать блоки так, чтобы заполнять ямы или активировать кнопки.

5. **"Сбор комбинаций" (Match-3 path):**
   * Игрок движется по полю, заполненному цветными тайлами.
   * Задача: наступать подряд на 3 тайла одного цвета, чтобы получить бонус.

## 5 тематик для применения

1. **Dungeon Crawler (Подземелье):**
   * Пиксель-арт, рыцарь, скелеты, сундуки.
   * Классика жанра, интуитивно понятные опасности (шипы) и награды (золото).

2. **Cyberpunk Hacker (Взлом сервера):**
   * Герой — программа-взломщик, поднимающаяся по структуре данных ("Башня").
   * Препятствия: файрволы, вирусы. Бонусы: пакеты данных, ключи шифрования.

3. **Infinite Garden (Волшебный боб):**
   * Мы управляем ростком, который тянется к солнцу.
   * Препятствия: камни, сухие ветки, жуки.
   * Цель: собирать капли росы и солнечный свет.

4. **Space Elevator (Космический лифт):**
   * Дроид-ремонтник ползет вверх по тросу космического лифта.
   * Опасности: космический мусор, метеориты, солнечная радиация.

5. **Mountain Climber (Альпинист):**
   * Суровое восхождение на Эверест.
   * Механика управления выносливостью и температурой.
   * Нужно выбирать надежные уступы и избегать ледяных участков.
