# Steppy Scroller: анализ прод‑гифки (2025‑12‑22)

Источник:
- Файл: `games/steppy-scroller/screenshots/gameplay-1766351943076-prod.gif`
- Разрешение: 600x1286, 8 fps, длительность ~5.12s

## Общее впечатление от геймплея
- Базовый цикл считывается сразу: 3 кнопки, движение вверх, счётчик высоты.
- Движение ощущается как прокрутка фона; у игрока нет читаемого шага/действия.
- Препятствия и бонусы близки по визуальному весу, угрозу не ощущаешь.
- Обратная связь по вводу видна на кнопках, но у персонажа она почти отсутствует.
- Темп ровный, но без «микро‑драмы» в момент выбора.

## Первые 8 кадров подряд (без пропусков)
1) frame01: Стартовое состояние. Пустая сетка, Height: 0m, игрок по центру внизу, кнопки без подсветки.
2) frame02: Подсвечена центральная кнопка (ввод «вперёд»). Сцена ещё не сдвинута.
3) frame03: Сдвиг поля вверх, Height: 1m. Появились объекты в верхних рядах (2 синих, 1 зелёный справа).
4) frame04: Центральная кнопка снова подсвечена (второй ввод). Сцена статична.
5) frame05: Сдвиг поля, Height: 2m. Объекты ближе: две синих слева, два зелёных справа, один синий в центре‑право.
6) frame06: Центральная кнопка подсвечена (третий ввод). Сцена статична.
7) frame07: Сдвиг поля, Height: 3m. Кластер справа приблизился к центру; синие слева выровнялись по вертикали.
8) frame08: Центральная кнопка подсвечена (четвёртый ввод). Сцена остаётся как в frame07, ждёт следующий сдвиг.

## Наблюдения
- Цикл стабилен: подсветка кнопки → следующий кадр даёт сдвиг и рост высоты.
- Персонаж не анимируется в момент хода, из‑за чего теряется ощущение «я сделал шаг».
- Объекты появляются без «телеграфирования», воспринимаются как внезапные блоки.

## Идеи для усиления ощущения игры (минимальные правки)
- Явный «шаг»: короткий сквош/пульс игрока в момент нажатия.
- Контраст опасностей: сделать препятствия темнее/фактурнее, добавить визуальный «опасный» акцент.
- Микро‑обратная связь: подсветка выбранной кнопки + лёгкий «подъём»/tween поля.
- Чуть больше «прогноза»: подсветить 2–3 ближайшие клетки пути, чтобы выбор ощущался осмысленнее.
- Привязать счётчик высоты к миру: фон/градиент меняется с ростом, а не просто цифра.
